# SmartHomeSystem
Modular smart home system with centralized communication, and visualization powered by ThingSpeak.

## Folder structure
* `src/` contains production source code used for the system
    * `src/modules/` contains project sketches for modules
* `old/` contains old sketches which may be useful for future reference
* `examples/` contains relevant examples of code usage, e.g. usage of the `sender_comm` module-side communication interface.
